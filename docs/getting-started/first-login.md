# First Login Setup

After your initial login, you'll need to complete some setup steps to personalize your Platform Console experience.

## Welcome Screen

After your first successful login, you'll see a welcome screen that guides you through the initial setup process.


## Profile Setup

### Step 1: Complete Your Profile

You'll be prompted to complete your profile information:


**Required Information:**

- **Full Name**: Your first and last name
- **Phone Number**: For account verification and notifications
- **Time Zone**: To display times correctly in your region
- **Language Preference**: Choose your preferred language

### Step 2: Profile Picture

Upload a profile picture (optional but recommended):

1. Click on the placeholder profile icon
2. Select "Upload Image"
3. Choose a photo from your device
4. Crop and adjust as needed
5. Click "Save"


## Organization Setup

### Joining Your Organization

If you were invited to join an existing organization:

1. You'll see a notification about your organization invitation
2. Click "Accept Invitation"
3. Review the organization details
4. Confirm your role and permissions


### Creating a New Organization (Admin Users)

If you're setting up a new organization:


**Organization Details:**

- **Organization Name**: Your company or team name
- **Organization Type**: Select from available options
- **Industry**: Choose your industry category
- **Size**: Number of team members

## Security Setup

### Two-Factor Authentication (2FA)

For enhanced security, you may be required to set up 2FA:


**Setup Steps:**

1. Download an authenticator app (Google Authenticator, Authy, etc.)
2. Scan the QR code with your app
3. Enter the 6-digit verification code
4. Save your backup codes in a safe place

## Initial Configuration

### Notification Preferences

Configure how you want to receive notifications:


**Options:**

- **Email Notifications**: System alerts via email
- **In-App Notifications**: Alerts within the platform
- **SMS Notifications**: Critical alerts via text
- **Frequency**: How often to receive digest emails

### API Access

If you need API access, generate your first API key:


**Steps:**

1. Go to "API Keys" section
2. Click "Generate New Key"
3. Give it a descriptive name
4. Set permissions and scope
5. Copy and securely store the key

## Common First-Time Issues

### Profile Picture Not Uploading

- **Check file size**: Images should be under 5MB
- **Check file format**: Use JPG, PNG, or GIF formats
- **Try a different browser**: Some browsers have upload restrictions

### Organization Invitation Not Received

- **Check spam folder**: Invitations might be filtered
- **Verify email address**: Ensure the correct email was used
- **Contact administrator**: Ask them to resend the invitation

### 2FA Setup Problems

- **Time synchronization**: Ensure your device time is correct
- **App compatibility**: Use supported authenticator apps
- **Backup codes**: Always save your backup codes safely
